Button

Nút bấm dùng class nano-button. Mỗi kiểu màu có ba biến thể: filled, outline (--outline), ghost (--ghost). Thêm --active để giữ trạng thái active, thuộc tính disabled để vô hiệu hóa.

Màu semantic: --primary, --danger, --warning, --success, --secondary. Màu theme: --blue, --red, --orange, --yellow, --green, --indigo, --violet.

1. Default button

Snippets
HTML
<button type="button" class="nano-button">Default</button>
<button type="button" class="nano-button nano-button--active">Active</button>
<button type="button" class="nano-button" disabled>Disabled</button>
HTML
<button type="button" class="nano-button nano-button--outline">Outline</button>
HTML
<button type="button" class="nano-button nano-button--ghost">Ghost</button>

2. Primary button

Snippets
HTML
<button type="button" class="nano-button nano-button--primary">Primary</button>
<button type="button" class="nano-button nano-button--primary-outline">Outline</button>
<button type="button" class="nano-button nano-button--primary-ghost">Ghost</button>

3. Danger, Warning, Success, Secondary

Snippet
HTML
<button class="nano-button nano-button--danger">Danger</button>
<button class="nano-button nano-button--warning">Warning</button>
<button class="nano-button nano-button--success">Success</button>

4. Màu theme (blue, red, orange, …)

Snippet
HTML
nano-button--blue | nano-button--red | nano-button--orange | nano-button--yellow
nano-button--green | nano-button--indigo | nano-button--violet
