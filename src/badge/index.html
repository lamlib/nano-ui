Badge

Badge dùng để hiển thị nhãn nhỏ (trạng thái, số thông báo, tag). Dùng class nano-badge kết hợp modifier màu: nano-badge--primary, --success, --warning, --danger, --secondary.

Thêm nano-badge--outline để dùng kiểu viền (outline).

1. Màu

primary success warning danger secondary
Snippets
HTML
<span class="nano-badge nano-badge--primary">primary</span>
HTML
<span class="nano-badge nano-badge--success">success</span>
<span class="nano-badge nano-badge--warning">warning</span>
<span class="nano-badge nano-badge--danger">danger</span>
<span class="nano-badge nano-badge--secondary">secondary</span>

2. Số (notification)

3 12 99+
Snippet
HTML
<span class="nano-badge nano-badge--danger">3</span>
<span class="nano-badge nano-badge--primary">12</span>
<span class="nano-badge nano-badge--success">99+</span>

3. Outline

New Active Pending Closed Draft
Snippet
HTML
<span class="nano-badge nano-badge--outline nano-badge--primary">New</span>
<span class="nano-badge nano-badge--outline nano-badge--success">Active</span>
<span class="nano-badge nano-badge--outline nano-badge--danger">Closed</span>

4. Kèm chữ

Thông báo 5 mới. Trạng thái: Đã duyệt.

Snippet
HTML
Thông báo <span class="nano-badge nano-badge--primary">5</span> mới. Trạng thái: <span class="nano-badge nano-badge--success">Đã duyệt</span>.
Class Mô tả
nano-badgeBase: inline-flex, padding, font-size, border-radius
nano-badge--primaryMàu primary
nano-badge--successMàu success
nano-badge--warningMàu warning
nano-badge--dangerMàu danger
nano-badge--secondaryMàu secondary
nano-badge--outlineKiểu viền (kết hợp với --primary, --success, …)
