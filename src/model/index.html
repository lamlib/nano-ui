Model (Modal)

Model dùng thẻ dialog HTML5, class nano-model, kèm data-nano-component="model". Nội dung chia thành nano-model__body và nano-model__footer (nút hành động).

Mở/đóng bằng JS: nút có data-nano-trigger="model", data-nano-target="id-dialog", data-nano-action="open" hoặc close. Cần load nano.min.js.

1. Model cơ bản

Snippet
HTML
<dialog id="my-model" data-nano-component="model" class="nano-model">
  <div class="nano-model__body">
    <h3 class="nano-text-lg nano-font-semibold nano-mb-4">Tiêu đề</h3>
    <p>Nội dung model.</p>
  </div>
  <div class="nano-model__footer">
    <button type="button" data-nano-trigger="model" data-nano-target="my-model" data-nano-action="close" class="nano-button nano-button--outline">Hủy</button>
    <button type="button" data-nano-trigger="model" data-nano-target="my-model" data-nano-action="close" class="nano-button nano-button--primary">Xong</button>
  </div>
</dialog>
<button type="button" data-nano-trigger="model" data-nano-target="my-model" data-nano-action="open" class="nano-button nano-button--primary">Mở model</button>

4. Trigger qua JavaScript

Model dùng thẻ dialog HTML5 nên có thể mở/đóng trực tiếp bằng JavaScript qua API native: showModal() để mở (có backdrop, bắt focus), close() để đóng. Không bắt buộc phải dùng data-nano-trigger.

Snippet
JavaScript
// Lấy tham chiếu dialog (theo id hoặc selector)
const dialog = document.getElementById('my-model');
// Hoặc: document.querySelector('[data-nano-component="model"]');

// Mở model (hiển thị backdrop, focus vào dialog)
dialog.showModal();

// Đóng model
dialog.close();

// Lắng nghe sự kiện đóng (khi user đóng hoặc gọi close())
dialog.addEventListener('close', function() {
  console.log('Model đã đóng');
});
HTML + JavaScript
<button type="button" id="btnOpen" class="nano-button nano-button--primary">Mở model</button>
<dialog id="my-model" data-nano-component="model" class="nano-model">...</dialog>

<script>
document.getElementById('btnOpen').addEventListener('click', function() {
  document.getElementById('my-model').showModal();
});
</script>
Class / Attribute Mô tả
nano-modelClass đặt trên dialog (width, margin, border-radius, backdrop)
nano-model__bodyVùng nội dung chính (padding)
nano-model__footerVùng chân (nút Hủy / Xác nhận, flex end)
data-nano-component="model"Đánh dấu dialog do Nano JS xử lý mở/đóng
data-nano-trigger="model"Nút hoặc element kích hoạt (cùng với target + action)
data-nano-target="id-dialog"ID của dialog cần mở/đóng
data-nano-action="open" / closeHành động: mở hoặc đóng model
closedby="any"Tùy chọn: đóng khi click bên ngoài (nếu JS hỗ trợ)
dialog.showModal()Mở model (backdrop, focus). Gọi trực tiếp trên element dialog.
dialog.close()Đóng model. Có thể truyền giá trị trả về: dialog.close(value).
