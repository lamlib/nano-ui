Dropdown

Menu thả xuống: trigger (nút hoặc link) bên ngoài, menu ẩn trong nano-dropdown__menu. Mỗi mục dùng nano-dropdown__item (thẻ a hoặc button). Chia nhóm bằng nano-dropdown__divider.

Menu hiện khi hover vào vùng nano-dropdown, hoặc khi thêm class nano-dropdown--open (có thể toggle bằng JS). Căn menu bên phải trigger: thêm nano-dropdown__menu--right lên menu.

1. Dropdown cơ bản

Hành động 1 Hành động 2 Hành động 3
Snippet
HTML
<div class="nano-dropdown">
  <button type="button" class="nano-button nano-button--primary">Dropdown</button>
  <div class="nano-dropdown__menu">
    <a href="#" class="nano-dropdown__item">Hành động 1</a>
    <a href="#" class="nano-dropdown__item">Hành động 2</a>
  </div>
</div>

2. Divider và item danger

Chỉnh sửa Sao chép
Snippet
HTML
<div class="nano-dropdown__menu">
  <a href="#" class="nano-dropdown__item">Chỉnh sửa</a>
  <div class="nano-dropdown__divider"></div>
  <button type="button" class="nano-dropdown__item nano-dropdown__item--danger">Xóa</button>
</div>

3. Menu căn phải

Option A Option B
Snippet
HTML
<div class="nano-dropdown__menu nano-dropdown__menu--right">
  <a href="#" class="nano-dropdown__item">Option A</a>
  <a href="#" class="nano-dropdown__item">Option B</a>
</div>

4. Mở bằng class (JS)

Mặc định menu hiện khi hover. Nếu cần mở/đóng bằng click (ví dụ trên mobile), thêm hoặc xóa class nano-dropdown--open trên wrapper nano-dropdown. Nano UI không kèm sẵn JS cho dropdown; bạn tự gắn click vào trigger để classList.toggle('nano-dropdown--open').

Class Mô tả
nano-dropdownWrapper (position: relative, display: inline-flex). Trigger + menu đặt bên trong.
nano-dropdown__menuMenu thả xuống (absolute, ẩn mặc định; hiện khi hover hoặc parent có --open)
nano-dropdown__menu--rightCăn menu sát cạnh phải trigger (right: 0)
nano-dropdown__itemMỗi mục menu (link hoặc button), block, padding
nano-dropdown__item--dangerItem nguy hiểm (hover: nền đỏ nhạt, chữ danger)
nano-dropdown__dividerĐường kẻ ngăn cách giữa các nhóm item
nano-dropdown--openModifier trên wrapper: giữ menu hiển thị (dùng khi điều khiển bằng click/JS)
