Code

Hiển thị code inline và khối code: nano-code (inline), nano-pre (khối), nano-code-block (khối kèm nút Copy). Có thể dùng thêm nano-var (biến), nano-kbd (phím), nano-samp (output mẫu).

Để tô màu cú pháp (syntax highlighting), dùng plugin Highlight.js trong thư mục plugin/highlight/: load highlight.min.js và một theme (vd. styles/vs.min.css), thêm class language-xxx lên pre hoặc code, rồi gọi hljs.highlightAll().

1. Inline code, biến, phím

Đây là biến x và inline code console.log('hello'). Nhấn Ctrl + C để copy.

Snippets
HTML
<code class="nano-code">value</code>
<var class="nano-var">x</var>
<kbd class="nano-kbd">Ctrl</kbd> + <kbd class="nano-kbd">C</kbd>

2. Khối code + nút Copy

JavaScript
function hello() {
  console.log('Hello, Nano UI');
  return true;
}
Snippet
HTML
<div class="nano-code-block">
  <h6 class="nano-code-block__title">JavaScript</h6>
  <button type="button" class="nano-code-block__copy" data-nano-trigger="copy-code" aria-label="Copy">Copy</button>
  <pre class="nano-pre"><code>your code here</code></pre>
</div>

3. Output mẫu (samp)

Terminal:

$ npm install Your content is copied! { "status": "ok" }
Snippet
HTML
<div class="nano-code-block">
  <button type="button" class="nano-code-block__copy" data-nano-trigger="copy-code" aria-label="Copy">Copy</button>
  <samp class="nano-samp">Output or terminal text here</samp>
</div>

4. Plugin Highlight.js (syntax highlighting)

Thư mục plugin/highlight/ chứa Highlight.js. Load file JS và một theme CSS, thêm class language-javascript, language-css, language-html, … lên thẻ code bên trong pre, sau đó gọi hljs.highlightAll() để tô màu.

JavaScript
const name = 'Nano UI';
function greet() {
  return `Hello, ${name}`;
}
Snippet
HTML + Script
<link rel="stylesheet" href="plugin/highlight/styles/vs.min.css">
<pre class="nano-pre"><code class="language-javascript">const x = 1;</code></pre>
<script src="plugin/highlight/highlight.min.js"></script>
<script>hljs.highlightAll();</script>
Class / Element Mô tả
nano-codeCode inline (nền xám nhạt, đậm)
nano-varBiến (italic, đậm), dùng với <var>
nano-kbdPhím bấm (dạng nút), dùng với <kbd>
nano-preKhối code (white-space: pre, nền xám nhạt)
nano-sampOutput mẫu / terminal (nền tối, chữ sáng)
nano-code-blockWrapper khối code (position relative, chứa copy + pre/samp)
nano-code-block__titleTiêu đề khối (tùy chọn), góc trái trên
nano-code-block__copyNút Copy, cần data-nano-trigger="copy-code" và nano.min.js
language-xxxClass cho Highlight.js (vd. language-javascript, language-css)
