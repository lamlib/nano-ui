Form

Form dùng nano-form bọc ngoài, mỗi nhóm label + input đặt trong nano-field. Input dùng class nano-input kèm modifier: nano-input-text, nano-input-email, nano-input-number, nano-input-textarea, nano-input-select, …

Thông báo gợi ý hoặc lỗi dùng nano-field-message. Trạng thái lỗi: thêm nano-field--error lên nano-field. Nút gửi/reset đặt trong nano-form-actions.

1. Form cơ bản

Snippet
HTML
<form class="nano-form" action="#" method="get">
  <div class="nano-field">
    <label for="name">Họ tên</label>
    <input id="name" class="nano-input nano-input-text" type="text" placeholder="Nhập họ tên">
  </div>
  <div class="nano-form-actions">
    <button type="submit" class="nano-button nano-button--primary">Gửi</button>
    <button type="reset" class="nano-button nano-button--outline">Đặt lại</button>
  </div>
</form>

2. Field message và trạng thái lỗi

Mô tả hoặc gợi ý cho người dùng.
Thông báo lỗi màu đỏ.
Snippet
HTML
<div class="nano-field">
  <label>Ô có mô tả</label>
  <input class="nano-input nano-input-text" type="text">
  <span class="nano-field-message">Mô tả hoặc gợi ý.</span>
</div>
<div class="nano-field nano-field--error">
  <label>Ô lỗi</label>
  <input class="nano-input nano-input-text" type="text">
  <span class="nano-field-message">Thông báo lỗi.</span>
</div>

</div>

</label>

6. Disabled

Class Mô tả
nano-formWrapper form (flex column, gap, max-width 28rem)
nano-fieldNhóm label + input (flex column, margin-bottom)
nano-inputBase cho input, select, textarea (border, padding, focus)
nano-input-text nano-input-email …Modifier theo type (text, email, number, password, tel, url, search, textarea, select)
nano-field-messageMô tả hoặc thông báo lỗi dưới input
nano-field--errorTrạng thái lỗi (border đỏ, message màu danger)
nano-form-actionsVùng nút (flex, gap) trong form
nano-fieldsetNhóm field (border, padding)
nano-legendTiêu đề fieldset
