Pagination

Pagination splits content or data into serveral pages, with a control for navigating to the next or previous page.

1. Basic

Previous 1 2 3 Next
Snippets
HTML
<nav class="nano-pagination" aria-label="Pagination">
    <a href="#" class="nano-pagination__item" aria-label="Previous page">
        Previous
    </a>
    <a href="#" class="nano-pagination__item">1</a>
    <a href="#" class="nano-pagination__item nano-pagination__item--active">2</a>
    <a href="#" class="nano-pagination__item">3</a>
    <a href="#" class="nano-pagination__item" aria-label="Next page">
        Next
    </a>
</nav>
